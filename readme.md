# 🃏 Solitaire (Patience)

**Solitaire**, also known as **Patience**, is a classic single-player card game that challenges your logic, focus, and strategic thinking. This timeless game has been enjoyed for generations, both digitally and with physical cards.

---

## 📌 Overview

The goal of Solitaire is simple yet engaging:

> Move all cards from the tableau to the foundation piles in the correct order.

Players must carefully organize cards using strategy and planning to successfully complete the game.

---

## 🧩 Gameplay

- The game uses a **standard 52-card deck**
- Cards are shuffled and dealt into **7 tableau columns**:
  - Column 1 → 1 card  
  - Column 2 → 2 cards  
  - ...  
  - Column 7 → 7 cards  

- In each column:
  - The **top card is face-up**
  - Remaining cards are **face-down**

- Additional elements:
  - **Foundation piles** (top area)
  - **Stock pile** (draw pile)

---

## 📜 Rules

### 🔹 Foundation Rules
- Build piles by suit:
  - Start with **Ace**
  - End with **King**
- Each suit has its own foundation pile

### 🔹 Tableau Rules
- Only **top cards** can be moved
- Cards must be stacked:
  - In **alternating colors** (red/black)
  - In **descending order** (K → Q → J → ...)

### 🔹 Empty Column Rule
- Only a **King** (and its stack) can fill an empty column

### 🔹 Stock Pile
- When no moves are available:
  - Draw cards from the **stock pile**
  - Continue gameplay

---

## 🎯 Objective

Win the game by:

- Moving **all cards** to the foundation piles
- Arranging them correctly by suit from **Ace → King**
